# Quick Fundz: Peer-to-Peer Investment and Loan Platform

![image](https://github.com/user-attachments/assets/8944814e-2e4e-4910-8e02-35f863d2e726)


Quick Fundz is a **peer-to-peer (P2P) investment and loan platform** that connects users directly, allowing them to agree on loans or investments at certified interest rates. With no brokers involved, Quick Fundz ensures a transparent, efficient, and user-friendly experience for both borrowers and investors. Built with **React.js**, **Tailwind CSS**, and **Aceternity UI**, this platform is designed for seamless financial transactions.

## Installation ğŸš€

Follow these steps to set up Quick Fundz locally:

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/alvin-dotcom/quick_fundz.git
   cd quick_fundz
   ```

2. **Install Dependencies**:
   ```bash
   npm install
   ```

3. **Run the Development Server**:
   ```bash
   npm start
   ```

4. **Open the App**:
   Visit `http://localhost:3000` in your browser to use Quick Fundz.
